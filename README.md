# 🤖 AI Chatbot

A simple AI Chatbot built using Python (Flask), HTML, CSS, and JavaScript.  
This chatbot can respond to basic user inputs and provides an interactive web interface.

 🚀 Features
- Simple and interactive chat interface
- Real-time responses using Flask backend
- Clean UI using HTML & CSS
- Handles basic conversations (greetings, etc.)

 🛠️ Tech Stack
- Python (Flask)
- HTML
- CSS
- JavaScript

 ▶️ How to Run
1. Install Flask:
   pip install flask

2. Run the application:
   python app.py

3. Open browser and go to:
   http://127.0.0.1:5000
